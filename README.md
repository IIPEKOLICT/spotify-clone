# spotify

### Environment variables

Backend

- `GREETINGS`

Frontend

### Clone repository

```shell
git clone git@github.com:not-yumasoft/spotify.git
cd spotify
npm i
npm run init
```

### Run backend

```shell
cd server
npm run server:dev
```

Backend will start at 5555 port (you can change it by providing custom PORT environment variable)

### Run frontend

```shell
cd client
npm run client:dev
```

Frontend will start at 3000 port
